# marvelapp

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

#Adicione keys fornecidas pela marvel no arquivo main.js

```
